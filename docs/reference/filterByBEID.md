# Filter an object to keep only a set of BEIDs

Filter an object to keep only a set of BEIDs

## Usage

``` r
filterByBEID(x, toKeep, ...)
```

## Arguments

- x:

  an object representing a collection of BEID (e.g. BEIDList)

- toKeep:

  a vector of elements to keep

- ...:

  method specific parameters
