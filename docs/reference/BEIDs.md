# Get the BEIDs from an object

Get the BEIDs from an object

## Usage

``` r
BEIDs(x, ...)
```

## Arguments

- x:

  an object representing a collection of BEID (e.g. BEIDList)

- ...:

  method specific parameters

## Value

A tibble with at least 4 columns:

- value

- be

- source

- organism

- ...
